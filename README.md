# 🛒 Market Analysis using SQL, Python & Power BI

## 📌 Overview
Analyzed a large-scale online grocery dataset to uncover customer purchasing behavior, product performance, reorder patterns, and demand trends. The project provides actionable business insights to optimize marketing strategies, inventory planning, and customer satisfaction. :contentReference[oaicite:0]{index=0}

→ Power BI Dashboard
→ Business Insights & Recommendations
